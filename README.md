# Отчёт о потреблении кофе

CLI-скрипт, который читает один или несколько CSV-файлов с данными студентов и строит отчёт `median-coffee` - медианную трату на кофе по каждому студенту по всем переданным файлам.

## Пример запуска

```bash
python main.py --files session1.csv --report median-coffee
python main.py --files session1.csv session2.csv --report median-coffee
```

## Пример вывода

```
+------------------+-----------------+
| student          |   median_coffee |
+==================+=================+
| Иван Кузнецов    |             650 |
+------------------+-----------------+
| Алексей Смирнов  |             500 |
+------------------+-----------------+
```

## Архитектура

Код разделён на несколько небольших модулей:

* `models` - доменная модель `StudyRecord`
* `reader` - чтение и валидация CSV-файлов
* `reports` - реализация отчётов
* `cli` - обработка аргументов командной строки и вывод результата

Отчёты реализованы как отдельные классы и подключаются через реестр (`app/reports/registry.py`). Это позволяет добавлять новые отчёты без изменения CLI - достаточно реализовать новый класс отчёта и зарегистрировать его.

Все отчёты наследуются от базового класса `BaseReport`, который задаёт общий интерфейс.

Основной функционал покрыт `pytest`-тестами.
