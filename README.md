# CoffeeStats CLI

`CoffeeStats CLI` - пет-проект для анализа трат на кофе из CSV-файлов.
Сейчас реализован отчёт `median-coffee`: медианная трата на кофе по каждому студенту с учётом всех переданных файлов.

## Возможности

- Чтение одного или нескольких CSV-файлов
- Валидация входных данных
- Расчёт медианы по каждому студенту
- Табличный вывод результата в консоль
- Расширяемая архитектура отчётов через реестр

## Быстрый старт

```bash
pip install -r requirements.txt
python main.py --files session1.csv --report median-coffee
python main.py --files session1.csv session2.csv --report median-coffee
```

## Пример вывода

```text
+------------------+-----------------+
| student          |   median_coffee |
+==================+=================+
| Иван Кузнецов    |             650 |
+------------------+-----------------+
| Алексей Смирнов  |             500 |
+------------------+-----------------+
```

## Архитектура

- `app/models.py` - доменная модель `StudyRecord`
- `app/reader.py` - чтение и валидация CSV
- `app/reports/` - отчёты и их регистрация
- `app/cli.py` - аргументы командной строки и вывод

Новые отчёты подключаются без изменения CLI: достаточно реализовать класс отчёта и зарегистрировать его в `app/reports/registry.py`.
